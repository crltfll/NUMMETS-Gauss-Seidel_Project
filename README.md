# NUMMETS-Gauss-Seidel_Project

This project is in partial requirement to the course Numerical Methods, and is an impplementation of the Gauss-Seidel Algorithm using the Python programming language in a Jupyter Notebook file for the processing of small-scale microgrids up to 6 buses.

Contained in the repository is the .ipynb file that encompasses the total code as well as the sample output for the proof of the code's successful execution. The proper documentation and paper project is also attached as a PDF.
